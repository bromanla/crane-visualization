# Crane Visualization

Визуализация положения крана по данным с энкодеров. В репозитории содержится скетч для Arduino Uno, Node JS приложение и веб-клиент.

## Communication

Микроконтроллер -> Приложение (serial port)

Приложение -> Веб-клиент (websocket)
